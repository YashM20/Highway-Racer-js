# Game Audio

This directory should contain the following audio files for the game:

1. crash.mp3 - Sound for collision with obstacles
2. engine.mp3 - Engine sound during gameplay
3. power_up.mp3 - Sound for collecting power-ups
4. lane_change.mp3 - Sound for changing lanes
5. game_music.mp3 - Background music during gameplay
6. menu_music.mp3 - Background music for the menu

You can create or find suitable audio files for these game sounds. The game has graceful error handling in case audio files are not available.
