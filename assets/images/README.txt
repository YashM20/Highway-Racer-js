# Game Images

This directory should contain the following image files for the game:

1. player_car.png - Player's car sprite
2. enemy_car.png - Enemy car obstacle
3. enemy_truck.png - Enemy truck obstacle
4. debris.png - Debris obstacle
5. shield.png - Shield power-up
6. clock.png - Slow motion power-up
7. heart.png - Extra life power-up
8. coin.png - Double score power-up
9. road_texture.png - Road texture
10. background.png - Game background

You can create or find suitable images for these game elements. The game has fallback rendering in case images are not available.
